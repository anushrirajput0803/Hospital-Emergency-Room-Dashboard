# Hospital-Emergency-Room-Dashboard
Excel dashboard analyzing hospital emergency room data
This project is an interactive Excel dashboard analyzing hospital emergency room data.

## Key Insights
- Total Patients
- Average Waiting Time
- Admission Rate
- Gender Distribution
- Department-wise Analysis

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning
